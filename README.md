# The Best Fishing App Ever Made

## Setup
1. `npm install`
2. Create a `.env` file:
   ```
   VITE_SUPABASE_URL=your-url
   VITE_SUPABASE_ANON_KEY=your-key
   VITE_MAPBOX_TOKEN=your-mapbox-token
   ```
3. `npm run dev`

## Features
- Log your fishing catches
- View map of locations
- Upload images, rate your catch
